Para exucutar o programa basta apenas seguir os seguintes comandos:

[1] Abrir o terminal
[2] Selecionar o diretório onde se encontra o código python a ser executado
[3] Excutar o comando: python trabalho1.py
[4] Inserir o nome do arquivo "netlist". OBS: Caso o mesmo não esteja no diretório do código a ser executado é necessário indicar o caminho completo.
[5] Caso deseje ler uma nova "netlist" apertar a tecla y (minúscula) e executar o passo 4 novamente.

Para sair do programa basta interromper sua execução apertando as teclas "ctrl+c" ou no passo [5] apertar qualquer tecla diferente de y.
